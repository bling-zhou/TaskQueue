# TaskQueue  
一个比较轻型的本地延迟队列的实现  
这个class可以解决的目标有:  
1.可以获得任意任务的剩余时间  
2.可以选择两种持久化的任务方式  
3.有着较强的可扩展性  
4.定时任务的维护是采用一个线程管理多个任务的方式,不管有多少个定时任务可以是由一个线程来进行管理维护,系统的开销会比较小  
5.设计的初衷是为了解决订单的超时取消任务  

项目的目录结构:  
1.main里面是设计的代码  
2.test里面有着相关的测试代码，主要是在本地测试,这个可以自己组装到spring里面  
3.image,相关的图片文件.
